API Timestamp microservice

Instructions

Enter a natural formatted date or a unix code date in the url and both the natural and unix code for that date will be returned.

An invalid date or format will return null.

Example usage:

https://kiwilifter-timestamp-api.herokuapp.com/December 15, 2015
https://kiwilifter-timestamp-api.herokuapp.com/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
